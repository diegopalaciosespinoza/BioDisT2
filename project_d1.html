Diseño Mecánico

Sensor MAX30100

Consta de 14 puertos, con unas dimensiones de 14x17mm . Integra un pulsioxímetro y cuenta con una salida para un monitor de frecuencia cardiaca. Posee dos Leds: un led rojo (660nm) y un led infrarrojo (920nm), un fotodetector, óptica especializada, filtro de luz ambiental entre 50 y 60Hz, y un conversor ADC delta sigma de 16 bits y de hasta 1000 muestras por segundo. Además posee un sensor de temperatura interno para compensar los efectos de la temperatura en la medición

Sensor DS18b20

Consta con una varilla metálica cubierta de acero inoxidable de 5mm de diámetro y 30 mm de largo . De fácil uso con microcontroladores como Arduino, PIC, ESP8266/ESP32. El sensor incluye internamente toda la electrónica necesaria: sensado, acondicionamiento, conversión analógica a digital e interfaz digital. Permite conocer la temperatura utilizando solo 1 cable para datos (1-Wire). Permite utilizar simultáneamente varios sensores en el mismo bus ya que cada sensor tiene un identificador único de fábrica. Peso del sensor 23 gramos.

Sensor Mpx2050

Con un tamaño de 2.60cm. Es un sensor de presión de silicio en chip de puerto doble con calibración y compensación de temperatura. El sensor de presión piezorresistivo de silicio de la serie MPX2050 proporciona una salida de tensión lineal muy precisa, directamente proporcional a la presión aplicada. El sensor es un diafragma de silicio monolítico simple con manómetro y una red de resistencias de película delgada en el chip. El sensor está ajustado por láser para precisión de alcance, calibración de desviación y compensación de temperatura.

Sensor de humedad DHT11

Es un sensor digital de temperatura y humedad relativa de bajo costo y fácil uso. Integra un sensor capacitivo de humedad y un termistor para medir el aire circundante, y muestra los datos mediante una señal digital en el pin de datos (no posee salida analógica).

Fuente de alimentación

El arduino se alimentará con una batería de 9V el cual va conectado al arduino mediante un cable adaptador, este será recargable.
